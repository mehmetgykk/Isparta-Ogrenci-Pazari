# Isparta Pazarı - Local Marketplace & Admin Dashboard
This project is a web-based marketplace application designed for local commerce in Isparta. It features a robust administration system where users and advertisements can be managed efficiently.

## ✨ Key Features
- **Admin Dashboard:** A centralized panel for managing listings and user activities.
- **Advertisement Management:** Tools for creating, highlighting, and organizing local ads.
- **User Interface:** Clean and user-friendly design for browsing local products.

## 🛠 Tech Stack
- **Backend:** C# / .NET (veya hangi dili kullandıysan)
- **Frontend:** HTML, CSS, JavaScript (Dashboard UI enhancements)
- **Database:** Microsoft SQL Server (SSMS)
